# Портфолио веб-разработчика — Спецификация

## 1. Обзор проекта
- **Название**: Developer Portfolio
- **Тип**: Одностраничный лендинг (SPA)
- **Технологии**: Vue 3 + Vite + Tailwind CSS (локально)
- **Деплой**: GitHub Pages

## 2. Структура страницы

### Секции (сверху вниз):
1. **Hero** — имя, должность, краткое описание, CTA-кнопка
2. **Обо мне** — краткая биография, навыки
3. **Проекты** — слайдер/карусель с карточками проектов
4. **Контакты** — email, ссылки на соцсети

### Навигация:
- Фиксированное меню сверху (скролл к секциям)
- Плавная прокрутка (smooth scroll)

## 3. UI/UX Спецификация

### Цветовая палитра
- **Фон**: `#0a0a0a` (почти чёрный)
- **Поверхность**: `#141414` (тёмно-серый для карточек)
- **Акцент**: `#e5e5e5` (светло-серый для текста)
- **Вторичный**: `#737373` (серый для описаний)
- **Подсветка**: `#ffffff` (белый для важных элементов)

### Типографика
- **Шрифт**: "Inter" (системный, без внешних загрузок через font-display: swap)
- **Заголовки**: 48px (hero), 32px (секции), 20px (карточки)
- **Тело**: 16px, line-height 1.6

### Компоненты

#### Меню навигации
- Прозрачный фон с blur
- Ссылки: Home, About, Projects, Contact
- Состояние: по умолчанию — серый, при наведении — белый

#### Hero секция
- Центрированный контент
- Имя крупно (48-64px)
- Должность ниже (20px, серый)
- Кнопка "View Projects" — тонкая рамка, при наведении заливка

#### Секция "Обо мне"
- Текст слева, навыки справа/снизу
- Теги навыков:pill-формы, тёмный фон

#### Секция проектов — Слайдер
- Горизонтальный скролл (snap scroll)
- Карточки: 320px ширина, компактные
- Карточка: изображение (placeholder), название, технологии, ссылка
- Скролл колёсиком по горизонтали + drag
- Кнопки навигации по бокам

#### Контакты
- Email крупно
- Иконки соцсетей (текстовые ссылки вместо иконок)
- Минималистично

### Анимации (Scroll-driven)
- Элементы появляются при скролле (fade + translateY)
- Плавные переходы (0.6s ease-out)
- Progress bar скролла (опционально)
