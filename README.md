# Портфолио веб-разработчика

Одностраничный лендинг с минималистичным дизайном, scroll-driven анимациями и автоматическим переключением темы.

## Технологии

- **HTML5** — семантическая разметка
- **Tailwind CSS v3** — стилизация (локальная установка)
- **Vanilla CSS** — scroll-driven анимации, CSS Scroll Snap для слайдера
- **Без JavaScript** для переключения темы (только CSS `prefers-color-scheme`)

## Структура проекта

```
portfolio/
├── index.html          # Главная страница
├── src/
│   └── input.css       # Исходный CSS с директивами Tailwind
├── dist/
│   └── output.css      # Скомпилированный CSS
├── tailwind.config.js  # Конфигурация Tailwind
├── postcss.config.js   # Конфигурация PostCSS
├── package.json
└── KODA.md             # Техническое задание
```

```bash
npm install
```

```bash

npm run watch
```


## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python3 -m http.server 8080

# Затем откройте http://localhost:8080
```

## Особенности

### 🎨 Автоматическая тема

Тема переключается автоматически на основе системных настроек через CSS `prefers-color-scheme`. JavaScript не используется.

### 📱 Слайдер портфолио

Реализован на **CSS Scroll Snap**:
- Плавная прокрутка с «прилипанием» слайдов
- Поддержка свайпов на мобильных
- Без JavaScript
- Адаптивная ширина слайдов (100% мобильный, 50% планшет, 33% десктоп)

### ✨ Scroll-driven анимации

Используются современные CSS анимации с `animation-timeline: view()`:

| Класс | Эффект |
|-------|--------|
| `.animate-fade-up` | Плавное появление с движением вверх |
| `.animate-scale-in` | Увеличение масштаба при появлении |
| `.animate-slide-left` | Выезд слева при скролле |
| `.read-progress` | Прогресс-бар чтения страницы |
| `.hero-bg` | Parallax эффект для фона Hero-секции |

**Поддержка браузеров:** Chrome 115+, Edge 115+, Safari 17+. В старых браузерах анимации не работают (graceful degradation).

### 🎯 Компоненты

- **Header** — фиксированная навигация с blur-эффектом
- **Hero** — главный экран с градиентным текстом и CTA-кнопками
- **About** — секция с биографией и навыками
- **Portfolio** — слайдер с карточками проектов
- **Contact** — контактная информация с иконками
- **Footer** — подвал с навигацией

## Кастомизация

### Цветовая палитра

Измените CSS-переменные в `src/input.css`:

```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f8f9fa;
  --text-primary: #1a1a1a;
  --text-secondary: #4b5563;
  --accent: #3b82f6;
  --accent-hover: #2563eb;
}
```

### Контент

Отредактируйте `index.html`:
- Имя и должность в Hero-секции
- Биография и навыки в разделе «Обо мне»
- Проекты в портфолио
- Контактная информация
